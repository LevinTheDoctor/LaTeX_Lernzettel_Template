\section{Cloud-Begriffe}
\subsection{Cloud Consumer}
Ein Cloud Consumer ist der Nutzer der Dienste, die eine Cloud bereitstellt. Er
entscheidet über seine eigenen Abonnements. Schließt man beispielsweise ein Abo
für Claude – den KI-Chatbot – ab, ist man ein Cloud Consumer. Dasselbe gilt für
iCloud, Spotify, Apple Music oder Netflix. Im Enterprise-Kontext sind typische
Abonnements Microsoft~365, Salesforce oder Verträge bei Cloud-Providern wie
Azure.
\subsection{Cloud Provider}
Ein Cloud Provider stellt die Infrastruktur und die Services einer Cloud
bereit. Darüber hinaus ist er für Betrieb und Sicherheit der Dienste
verantwortlich. Beispiele sind die Hyperscaler AWS, Azure und Google Cloud,
aber auch europäische Anbieter wie Hetzner.
\subsection{Cloud Auditor}
Ein Cloud Auditor prüft die Rechenzentren von Cloud Providern – insbesondere,
ob diese Datenschutz- und Compliance-Vorgaben einhalten.
\subsection{Cloud Broker}
Ein Cloud Broker ist ein unabhängiger Anbieter, der zwischen Cloud Consumer und
Cloud Provider vermittelt. Ein Beispiel wäre ein Systemhaus, das Lizenzen einer
Cloud einkauft und sie samt Support und Konfiguration an Consumer
weiterverkauft.
\subsection{Cloud Carrier}
Ein Cloud Carrier ist für eine sichere und stabile Datenübertragung zwischen
Cloud Consumer und Cloud Provider verantwortlich.
